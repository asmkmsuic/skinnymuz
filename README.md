Что такое Skinnymuz?
------------------

Skinnymuz — это открытая, кроссплатформенная цифровая аудио-рабочая станция для делания музыки. Тут есть продвинутый Piano Roll, Beat Sequencer, Song Editor и Mixer для сочинения, аранжировки и сведения музыки. Внутри уже есть 15+ синтезаторных плагинов, плюс поддержка VST2 и SoundFont2.

Особенности
--------

* Song Editor для раскладки мелодий, сэмплов, паттернов и автоматизации.
* Pattern Editor для делания битов и паттернов.
* Удобный Piano Roll для редактирования паттернов и мелодий.
* Mixer с бесконечным числом каналов и любым количеством эффектов.
* Много мощных инструментов и эффектов уже из коробки.
* Полная пользовательская трековая автоматизация и автоматизация от компьютера.
* Совместимость со многими стандартами, типа SoundFont2, VST2 (инструменты и эффекты), LADSPA, LV2, GUS Patches и полный MIDI.
